EXCEL BASICS

Learn the fundamental structure of Excel workbooks

Beginner

Module 1: Workbook & Worksheet Basics

Excel is the world's leading spreadsheet application, used for data analysis, financial modeling, and reporting. Think of Excel as a giant grid of cells organized into rows and columns. Each file is a workbook that can contain multiple worksheets (like pages in a notebook).

Every Excel workbook follows the same basic structure. The Ribbon at the top contains all the tools you need. The grid consists of columns (labeled A, B, C...) and rows (numbered 1, 2, 3...). The intersection of a column and row is a cell, identified by its address (e.g., A1).

Key Points:

  • A workbook is the entire Excel file (.xlsx)
  • Worksheets are tabs within a workbook
  • Cells are identified by column letter and row number (e.g., A1)
  • The formula bar shows the content of the active cell
  • Formulas always start with an equals sign =
  • Use & to join text (concatenation)
  • Functions like LEN() perform calculations

Excel Foundation Explained

Excel is built on a grid system that dates back to the earliest spreadsheet programs. Understanding the relationship between workbooks, worksheets, rows, columns, and cells is essential before diving into formulas and analysis. Every cell can contain text, numbers, or formulas, and Excel recalculates formulas automatically when data changes.

The Ribbon interface organizes hundreds of commands into tabs like Home, Insert, Page Layout, Formulas, Data, Review, and View. Learning to navigate these tabs efficiently will speed up your work. The Name Box (left of the formula bar) shows the active cell address and can be used to name ranges for easier reference.

Workbook Structure

A workbook can contain many worksheets. You can add, delete, rename, and color-code worksheet tabs. This helps organize related data (e.g., separate sheets for each month).

Calculation Engine

Excel's calculation engine updates results instantly when you change input values. This makes it perfect for what-if analysis and dynamic models.

File Formats

Modern Excel files use the .xlsx format (XML-based). Older versions used .xls. Excel can also open CSV, text files, and connect to external databases.

Real-World Application

Every finance professional, data analyst, and business owner uses Excel daily. Budgets are built in Excel, sales reports are generated, and complex financial models forecast company performance. Mastering Excel basics is the first step toward data literacy and analytical roles.

CELLS & FORMATTING

Learn to enter data and format cells

Beginner

Module 2: Data Entry and Formatting

Entering data correctly and formatting cells makes your spreadsheets readable and professional. You can format numbers as currency, dates, percentages, and more. Cell formatting also includes fonts, colors, borders, and alignment.

Excel automatically recognizes certain data types. For example, if you enter "Jan 2025", Excel treats it as a date. You can then apply different date formats. Number formatting changes how values appear without changing the underlying value.

Key Points:

  • Format cells using Home tab > Number group
  • Common formats: General, Number, Currency, Accounting, Date, Percentage
  • Use Merge & Center for titles spanning multiple columns
  • Apply borders, fill colors, and text styles from Home tab
  • Wrap text and merge cells for better layout
  • Formatting doesn't change the actual cell value, only its appearance
  • Use Format Painter to copy formatting quickly

Visual Communication

Formatting transforms raw data into understandable information. Proper number formatting (currency, percentages) prevents misinterpretation. Conditional formatting can automatically highlight important values like top performers or budget overruns. Cell styles and themes ensure consistency across reports.

Excel also supports custom number formats for specialized needs like phone numbers, ZIP codes, or tracking numbers. Understanding formatting is essential for creating professional reports that communicate effectively.

Currency vs Accounting

Currency format places the dollar sign right next to the number. Accounting format aligns currency symbols and decimals in a column, which is preferred for financial statements.

Date Systems

Excel stores dates as serial numbers (1 = Jan 1, 1900). This allows date arithmetic: you can subtract dates to find days between.

Conditional Formatting

Highlight cells based on rules: greater than a threshold, duplicate values, top 10%, etc. This makes trends and outliers instantly visible.

Financial Reporting

Accountants use precise formatting in income statements and balance sheets. Currency formatting, underlines, double underlines for totals, and proper indentation are standard. Conditional formatting might highlight negative variances in red.

Learn to enter data efficiently and write basic formulas

Beginner

Module 3: Data Entry and Basic Formulas

Efficient data entry saves time and reduces errors. Excel offers features like AutoFill, Flash Fill, and data validation. Formulas are what make Excel powerfulâ€”they perform calculations using cell references, operators, and functions.

Formulas can include arithmetic operators (+ - * / ^), comparisons (= > < >= <= <>), and functions like SUM(), AVERAGE(), COUNT(). Cell references can be relative (A1), absolute ($A$1), or mixed (A$1, $A1).

Key Points:

  • Formulas always start with =
  • Use cell references (like A1) so formulas update automatically
  • $ locks a reference (absolute): $A$1 always points to A1
  • Common functions: SUM, AVERAGE, COUNT, MAX, MIN, IF
  • AutoFill (drag the fill handle) copies formulas and fills series
  • Flash Fill (Ctrl+E) intelligently fills data based on pattern
  • Use parentheses to control order of operations

The Power of Formulas

Formulas turn Excel from a simple grid into a dynamic calculation engine. By using cell references, you create models that update automatically when inputs change. This is the foundation of financial modeling, budgeting, and data analysis.

The IF function introduces logic: "if this condition is true, do this; otherwise, do that." Combined with other functions, you can build sophisticated decision-making tools.

Relative vs Absolute

Relative references change when copied (A1 becomes A2). Absolute references ($A$1) stay fixed. Mixed references lock only row or column. Crucial for complex formulas.

Formula Auditing

Trace Precedents/Dependents shows relationships between cells. Evaluate Formula steps through calculation. These tools help debug errors.

Array Formulas

Advanced formulas that perform multiple calculations on one or more items. Enter with Ctrl+Shift+Enter in older Excel; dynamic arrays in modern Excel simplify this.

Loan Calculator

A loan amortization schedule uses formulas with mixed references. Payment amount (PMT), interest calculation, principal reduction. Changing the interest rate instantly updates the whole schedule.

Learn to organize data with Excel tables and named ranges

Intermediate

Module 4: Excel Tables and Named Ranges

Excel Tables (created with Ctrl+T) convert a range of data into a structured, dynamic table. Tables automatically expand as you add data, and formulas use structured references (like =SUM(Table1[Sales])) that are easier to read.

Named ranges let you assign a meaningful name to a cell or range (e.g., =Price instead of =B2:B10). Names make formulas clearer and less error-prone, especially in large workbooks.

  • Tables automatically expand with new rows/columns
  • Structured references use table and column names

  • Named ranges can be used in formulas, data validation, and navigation
  • Names are absolute references by default

  • Tables are essential for dashboards and dynamic reports

Structured Data Management

Excel Tables transform raw data into a structured format that's ideal for analysis. Unlike ordinary ranges, tables know where their data endsâ€”formulas, charts, and PivotTables automatically include new data added to the table. This makes tables the foundation of dynamic, self-updating reports.

Named ranges improve formula readability and maintainability. Instead of deciphering =SUM($B$2:$B$100), you see =SUM(Sales). Names also make navigation easierâ€”use the Name Box dropdown to jump to any named range instantly.

Dynamic Ranges

Tables automatically resize. When you add a row, any formula referencing the table expands automatically. This is crucial for dashboards that need to stay up-to-date.

Chart Integration

Charts based on table data automatically include new data points. No need to manually update chart ranges.

Built-in Filtering

Tables come with filter arrows in the header row. You can quickly sort and filter data without extra steps.

Sales Dashboard

A sales dashboard often uses a table as the source data. PivotTables and charts built on that table automatically include new sales records as they're entered. Named ranges like "TopProduct" can be used in formulas to highlight best-sellers.

Module 5: Data Validation and Form Controls

Data validation restricts what users can enter into a cell, reducing errors. You can create drop-down lists, set number ranges, restrict dates, or create custom rules with formulas.

Form controls (like buttons, checkboxes, option buttons) from the Developer tab let you build interactive user interfaces. These can be linked to cells and used in formulas to create dynamic models.

Try It Yourself:

Excel Validation
Live Preview (simulated)

Data Validation rules are applied to cells. In a real workbook, you'd see dropdown arrows, error alerts, and input messages.

  • A1: Dropdown (Yes/No/Maybe)
  • B1: Only numbers 1-100 allowed
  • C1: Only dates in 2025 allowed
  • Column A: No duplicates allowed

Key Points:

  • Data Validation found in Data tab > Data Validation
  • Create drop-down lists from ranges or comma-separated values
  • Add input messages and error alerts to guide users
  • Form controls (Developer tab) include buttons, checkboxes, scroll bars
  • Link form controls to a cell to capture their state (e.g., checkbox linked to TRUE/FALSE)
  • Use linked cells in formulas to create interactive models
  • Protect sheets to prevent users from changing validation rules

Input Control and User Experience

Data validation ensures that the data entering your workbook meets quality standards. Drop-down lists eliminate typos, number limits prevent out-of-range values, and custom formulas can enforce complex business rules. This is essential when multiple people enter data into the same file.

Form controls take interactivity further. A scroll bar can adjust a parameter, and a button can run a macro. Combined with worksheet formulas, you can build simple dashboards that let users explore scenarios without editing formulas directly.

Preventing Errors

Validation stops invalid entries before they happen. For critical data like IDs or dates, this is invaluable. You can also circle invalid data after the fact (Data > Data Validation > Circle Invalid Data).

Interactive Controls

Use scroll bars, spinners, and option buttons to let users adjust assumptions visually. Link them to cells and use those cells in formulas for dynamic what-if analysis.

Worksheet Protection

After setting up validation and controls, protect the sheet to prevent users from changing formulas or validation rules. Only unlocked cells can be edited.

Expense Report Template

A company expense form might use drop-downs for categories (Meals, Travel, Supplies), date validation to ensure dates are within the current month, and a checkbox for "Receipt attached". A submit button could run a macro to save the data to a database.

EXCEL FUNCTIONS

Module 6: Essential Functions: VLOOKUP, IF, SUMIFS, etc.

Excel has hundreds of functions, but mastering a core set unlocks most data tasks. Lookup functions (VLOOKUP, HLOOKUP, XLOOKUP) find values in tables. Logical functions (IF, AND, OR) make decisions. Math functions (SUMIFS, COUNTIFS, AVERAGEIFS) sum or count with multiple conditions.

Text functions (LEFT, RIGHT, MID, CONCAT) manipulate strings. Date functions (DATEDIF, EOMONTH, NETWORKDAYS) handle dates. Learning these will let you clean, analyze, and summarize data efficiently.

Key Points:

  • VLOOKUP looks right; INDEX/MATCH or XLOOKUP are more flexible
  • IF can be nested for multiple conditions
  • SUMIFS, COUNTIFS, AVERAGEIFS handle multiple criteria
  • Text functions clean and extract parts of strings
  • Date functions are essential for time-based analysis
  • Function arguments are separated by commas (or semicolons depending on locale)
  • Use Formula Autocomplete (Ctrl+Shift+A) to see syntax

Unlocking Data Insights

Functions are Excel's superpower. While basic arithmetic handles simple tasks, functions like VLOOKUP let you merge data from different tablesâ€”like looking up product prices from a price list. SUMIFS and COUNTIFS let you answer questions like "How many sales were above $1000 in the East region?" without sorting or filtering manually.

Modern Excel includes dynamic array functions (FILTER, SORT, UNIQUE) that spill results into multiple cells automatically, making formulas even more powerful.

Lookup Functions

VLOOKUP (vertical) and HLOOKUP (horizontal) are classic. XLOOKUP (Excel 365) is superior: it looks both directions, defaults to exact match, and handles arrays.

Conditional Aggregation

SUMIF, SUMIFS, etc., let you sum based on criteria. They're like lightweight PivotTables in formula form.

Text Manipulation

Functions like TRIM, CLEAN, PROPER, SUBSTITUTE help standardize messy data imported from other systems.

Sales Report

Combine VLOOKUP to bring in product categories, SUMIFS to total sales by region, and IF to flag underperforming products. Text functions can format product codes. This creates a dynamic report that updates as new data arrives.

Charts turn numbers into visual stories. Excel offers a wide variety: column, bar, line, pie, scatter, and more specialized charts like waterfall, funnel, and maps. Choosing the right chart type is crucial for clear communication.

Formatting options let you customize every element: titles, axes, legends, data labels, colors. You can also create combination charts (e.g., columns with a line) and add trendlines or error bars.

PivotTables are Excel's most powerful feature for summarizing large datasets. With a few clicks, you can aggregate thousands of rows into meaningful totals, averages, percentages, and more. They're interactiveâ€”drag fields to rearrange the analysis.

Power Query (Get & Transform) is a data connection and transformation tool. It lets you import data from various sources, clean it, merge tables, and load the result into Excel. The steps are repeatableâ€”refresh to update with new data.

Data Summarization & Automation

PivotTables revolutionized data analysis. Before PivotTables, creating a summary report meant writing complex formulas. Now, anyone can explore data interactively. They're essential for business intelligence tasks.

Power Query automates data preparation. Instead of manually cleaning messy exports, you build a query once. When you get new data, just refreshâ€”all cleaning steps reapply. This saves hours of repetitive work.

Calculated Fields

Add your own formulas within a PivotTable, like profit margin = (Sales - Cost)/Sales. These behave like regular fields.

PivotCharts

ACCESSIBILITY

Learn to create accessible spreadsheets

Intermediate

Module 9: Creating Accessible Excel Workbooks

Accessibility ensures that people with disabilities can use your spreadsheets. This includes users who rely on screen readers, keyboard navigation, or high-contrast settings. Excel provides tools to make workbooks accessible.

Best practices: use clear headings, add alt text to images/charts, avoid blank rows/columns, use meaningful sheet names, and ensure sufficient color contrast. The Accessibility Checker (Review > Check Accessibility) helps identify issues.

Try It Yourself:

Accessibility Tips
Live Preview

Checklist displayed. In a real workbook, you'd run the Accessibility Checker.

Key Points:

  • Use table formatting for data ranges (Ctrl+T)
  • Always provide alt text for images, shapes, and charts
  • Avoid blank rows/columns within data (they break navigation)
  • Use descriptive hyperlink text, not "click here"
  • Ensure font size is at least 11pt for readability
  • Use the Accessibility Checker regularly
  • Test with keyboard only (no mouse)

Inclusive Design

Accessibility benefits everyone, not just users with disabilities. Clear headings make navigation easier for all. Good color contrast helps in bright sunlight. Screen reader compatibility ensures your workbooks can be used by the visually impaired, who may be colleagues or clients.

Many organizations require accessibility compliance (e.g., Section 508 in US government). Excel's built-in tools make it easier to meet these standards.

Visual Impairments

Screen readers announce cell content, table structure, and alt text. Proper structure (headers, no merged cells) is essential.

Motor Impairments

Some users cannot use a mouse. Ensure all functionality is available via keyboard: Tab to navigate, Enter to select, etc.

Low Vision

High contrast and scalable text (Zoom) help. Avoid relying on color alone (use icons or patterns as well).

Government Reporting

A government agency requires all submitted spreadsheets to be accessible. They run the Accessibility Checker and reject any with errors. Knowing how to create accessible workbooks is essential for compliance.

Module 10: Macros, VBA, and Advanced Techniques

Visual Basic for Applications (VBA) is Excel's programming language. With VBA, you can automate repetitive tasks, create custom functions, and build interactive tools. Macros are recorded actions that you can play back.

Advanced formula techniques include array formulas, dynamic arrays (Excel 365), and complex nested functions. Power Pivot and Data Model let you work with millions of rows. This module takes you from user to developer.

Key Points:

  • Enable Developer tab (File > Options > Customize Ribbon)
  • Record macros for simple automation (View > Macros > Record Macro)
  • Edit macros in the VBA editor (Alt+F11)
  • VBA can automate almost any Excel action
  • Use relative references when recording for flexible macros
  • Assign macros to buttons or shapes for easy access
  • Always save as .xlsm to keep macros

Automation and Customization

VBA turns Excel into a development platform. You can build complete applications within Excelâ€”data entry forms, reporting tools, even games. Macros save time on repetitive tasks like formatting, data cleaning, or generating reports.

Advanced formulas in modern Excel include dynamic arrays (functions that spill results). For example, =SORT(FILTER(A2:B100, B2:B100>100)) returns a sorted, filtered list automatically. No more array formula keystrokes.

Macro Recording

Record a macro while you perform actions. Excel writes the VBA code. You can then edit it to add logic or make it more general.

Power Pivot

For massive datasets, Power Pivot (an Excel add-in) lets you create data models with relationships and DAX formulas, handling millions of rows.

Security

Macros can contain viruses. Only enable macros from trusted sources. Digitally sign your own macros.

Automated Reporting

A financial analyst runs a monthly report: import data, clean, format, create charts, and email to stakeholders. A VBA macro does all this in one click. The macro even saves the file with the current date and sends via Outlook.
